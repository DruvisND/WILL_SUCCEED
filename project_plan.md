# Project 2 项目规划：月度反转策略研究

## 项目概述

基于 CSMAR 月个股回报率数据 (TRD_Mnth.csv)，系统性地研究月度反转策略，探索最优参数配置，评估策略可交易性。

## 核心目标

1. 构建完整的量化策略研究工作流
2. 掌握防未来函数泄露的核心原则
3. 系统性探索参数空间，找到最优反转策略配置
4. 评估策略在现实成本下的表现

## 工作流框架

```
数据 → 信号 → 组合 → 成本 → 评估 → 证据
```

## Week 2：参数探索

采用**单因子实验原则**：每次只改变一个参数，其他参数保持不变。

### 维度 1：形成期探索

| 形成期 | 预期特点 |
|--------|----------|
| 1个月 | 短期反转效应强，换手率高 |
| 3个月 | 中期反转，可能最优 |
| 6个月 | 平衡信号强度与换手率 |
| 12个月 | 长期动量效应开始显现 |

**固定参数**：TopK=50, 标准化=Raw, 再平衡=月度

### 维度 2：标准化方法

| 方法 | 公式 | 特点 |
|------|------|------|
| Raw | 原始信号 | 简单，可能被极值主导 |
| Rank | rank(pct=True) | 消除极值，信号在[0,1] |
| Z-score | (x-mean)/std | 保留相对强度 |
| Winsorization | 限制在5%-95% | 处理极端值 |

**注意**：对于TopK策略，标准化不改变选股结果。

### 维度 3：TopK 选择

| TopK | 集中度 | 分散化 |
|------|--------|--------|
| 20 | 高 | 低 |
| 50 | 中 | 中 |
| 100 | 中低 | 中高 |
| 200 | 低 | 高 |

### 维度 4：再平衡频率

| 频率 | 优点 | 缺点 |
|------|------|------|
| 月度M | 及时响应信号 | 换手率高 |
| 季度Q | 换手率低 | 可能错过机会 |

## Week 3：成本与证据

### 交易成本模型

```
Net Return = Gross Return - Turnover × Cost_bps
```

### 中国A股成本参考

| 成本类型 | 费率 | 说明 |
|----------|------|------|
| 佣金 | 2-5 bps | 双向收取 |
| 印花税 | 10 bps | 仅卖出时收取 |
| 总成本 | 10-20 bps | 散户级别 |

### 成本敏感性测试

测试成本水平：0, 10, 20, 50 bps

### 可交易性评估清单

1. **临界成本**：策略净收益降至0时的成本水平
   - \>50 bps：抗成本能力强 ✅
   - 20-50 bps：中等 ⚠️
   - \<20 bps：弱 ❌

2. **换手率**：
   - \<500%：低 ✅
   - 500-1000%：中 ⚠️
   - \>1000%：高 ❌

3. **容量**：TopK越多、市值越大，容量越大

## Week 5：中国版三因子模型 (CH-3)

### 核心文献

Liu, Stambaugh & Yuan (2019) - "Size and value in China", Journal of Financial Economics

### CH-3 vs FF-3 主要区别

| 方面 | FF-3 (美国) | CH-3 (中国) |
|------|-------------|-------------|
| 样本 | 全部股票 | 剔除最小30%壳资源股票 |
| 价值指标 | Book-to-Market (B/M) | Earnings-to-Price (EP) |
| 组合加权 | 市值加权 | 市值加权 |
| 无风险利率 | T-bill | 一年期存款利率 |

### 因子构造：2×3 分组法

**步骤1：剔除壳资源**
- 所有A股按市值排序
- 剔除市值最小的30%股票（这些公司约83%曾被用于借壳上市）
- 剩余70%股票用于因子构造

**步骤2：按市值分组**
- 小盘股 (S)：市值 < 中位数
- 大盘股 (B)：市值 ≥ 中位数

**步骤3：按EP分组**
- 价值股 (V)：EP前30%（高盈利被低估）
- 成长股 (G)：EP后30%（低盈利被高估）
- 中性股 (M)：EP中40%

**6个组合**：

| 组合 | 规模 | EP分组 |
|------|------|--------|
| S/V | 小盘 | 价值股 |
| S/M | 小盘 | 中性股 |
| S/G | 小盘 | 成长股 |
| B/V | 大盘 | 价值股 |
| B/M | 大盘 | 中性股 |
| B/G | 大盘 | 成长股 |

### 因子计算公式

**MKT（市值因子）**：
$$MKT = R_{70\%} - R_f$$

**SMB（规模因子）**：
$$SMB = \frac{S/V + S/M + S/G}{3} - \frac{B/V + B/M + B/G}{3}$$

**VMG（价值因子）**：
$$VMG = \frac{S/V + B/V}{2} - \frac{S/G + B/G}{2}$$

### EP处理规则

- 正EP：正常参与排序
- 负EP（亏损公司）：归入成长组(G)

### CH-3 因子统计特征（论文参考）

| 因子 | 月均收益 | 标准差 | t统计量 |
|------|----------|--------|---------|
| MKT | 0.66% | 1.56% | 7.48 |
| SMB | 1.03% | 2.61% | 6.98 |
| VMG | 1.14% | 2.42% | 8.32 |

## Week 6：扩展因子池与因子归因

### 常见扩展因子

| 因子 | 名称 | 构造方法 | 中国市场表现 |
|------|------|----------|--------------|
| MOM | 动量因子 | 做多过去12月赢家，做空输家 | ✗ 无效，甚至反向 |
| RMW | 盈利因子 | 做多高盈利，做空低盈利 | ✓ 有效 |
| CMA | 投资因子 | 做多低投资，做空高投资 | ✗ 效应不显著 |
| 低波动 | 低波动因子 | 做多低beta，做空高beta | ✓ 有效 |

### 中国市场异象有效性

| 异象 | 月均alpha | 显著性 | 说明 |
|------|-----------|--------|------|
| 规模效应 | 0.97% | * | 小盘溢价显著 |
| EP价值效应 | 1.37% | *** | 最有效的价值指标 |
| ROE盈利效应 | 0.93% | ** | 盈利效应显著 |
| 低波动 | 1.03% | ** | 低波动异象 |
| 短期反转 | 1.49% | *** | 反转效应最强 |
| 换手率 | 1.27% | *** | 低换手股收益更高 |
| 动量 | - | - | 无效，甚至反向 |

### 因子归因方法

将策略收益对因子暴露进行回归：
$$R_i - R_f = \alpha + \beta_{MKT}MKT + \beta_{SMB}SMB + \beta_{VMG}VMG + \varepsilon$$

解读：
- **alpha**：策略无法被CH-3解释的超额收益
- **beta**：策略对各因子的暴露（风格暴露）
- 如果alpha不显著，说明策略只是风格暴露，非真正的alpha

### 可复述结论模板

```markdown
## 因子归因结论

**策略**：X个月反转策略，TopK=K

**回归结果**：
- alpha: XX bps/月 (t = X.XX)
- MKT beta: X.XX (t = X.XX)
- SMB beta: X.XX (t = X.XX)
- VMG beta: X.XX (t = X.XX)
- $R^2$: XX%

**结论**：
- [若alpha显著] 策略存在显著的独立alpha，无法被CH-3解释
- [若alpha不显著] 策略收益主要来自风格暴露（如做多小盘/价值股）
```

## 注意事项

1. **防泄露是核心纪律**：任何信号构造都不能使用未来数据
2. **单因子实验原则**：准确评估每个参数的独立影响
3. **成本检验可行性**：在现实成本下评估策略是否仍有竞争力
4. **实验记录规范**：确保可复现，便于团队协作
