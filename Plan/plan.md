# 基于社交媒体情绪分析的量化投资策略汇报方案

## 1. 研究问题（2分钟）

**研究问题**：基于社交媒体投资者情绪的量化投资策略研究

**为什么有趣**：
- 传统量价/财务数据因子日益拥挤，收益降低
- 社交媒体数据包含大量投资者情绪信息，可能提前反映市场变化
- 学术意义：验证情绪因子对股票收益率的预测能力
- 实践价值：为量化投资提供新的alpha来源

**预期发现**：
- 社交媒体情绪因子与股票未来收益率存在显著相关性
- 基于情绪因子的投资策略能够获得超额收益
- 情绪因子具有独立于传统因子的解释能力

## 2. 当前进展（4分钟）

**数据**：
- 数据源：微博、雪球、股吧
- 样本期间：2023年1月-2025年6月
- 数据获取：API和网络抓取
- 数据处理：文本清洗、分词、去停用词

**因子构造**：
1. **情感得分（Sentiment Score）**：使用BERT模型对帖子进行情感分析，计算每日情绪得分
2. **讨论热度（Attention）**：计算股票相关帖子的数量、转发量和评论量
3. **情绪变化率（Sentiment Change）**：计算情绪得分的日变化率
4. **情绪一致性（Sentiment Consistency）**：计算情绪得分的标准差，衡量市场观点的一致性

**初步结果**：
- 回测结果：年化收益率12.5%，夏普比率1.8，最大回撤15%
- 多空策略：多头组合收益率显著高于空头组合
- FF3归因：alpha为0.008（t=3.25，p<0.01），显著为正
- 因子相关性：情绪因子与传统因子（规模、价值、动量）相关性较低

**FF3归因**：
- 市场因子（MKT）：系数0.95，说明策略系统性风险接近市场
- 规模因子（SMB）：系数-0.12，说明策略偏好大盘股
- 价值因子（HML）：系数0.08，说明策略轻微偏好价值股
- Alpha：0.008，说明策略具有显著的超额收益

## 3. 遇到的问题（2分钟）

**技术问题**：
- 数据获取：部分平台API限制，网络抓取面临反爬措施
- 数据清洗：社交媒体数据噪音大，包含大量垃圾信息和水军
- 计算瓶颈：情感分析需要大量计算资源，实时性挑战

**方法问题**：
- 情绪分析准确度：中文NLP情感分析存在歧义，多义词语义难以准确识别
- 因子滞后性：情绪变化到股价反应可能存在时滞
- 样本偏差：社交媒体用户可能不能代表整体投资者群体

**结果问题**：
- 因果关系：情绪变化与股价变动的因果方向难以确定
- 过拟合风险：参数优化可能导致样本内表现好，样本外表现差
- 极端事件影响：市场剧烈波动时情绪因子可能失效

## 5. 传统数据准备（不考虑另类数据）

### 5.1 数据类型
传统量化策略测试通常依赖以下几类核心数据：
- **量价数据**：股票的开盘价、收盘价、最高价、最低价、成交量、成交额等，是计算收益率、波动率、动量等因子的基础。
- **财务数据**：公司的季度/年度财务报表数据，如营业收入、净利润、ROE、资产负债率、现金流等，用于构建价值、质量等因子。
- **市场基准数据**：如沪深300、中证500等指数的收益率数据，用于计算超额收益和进行基准对比。
- **风险因子数据**：如市场因子（MKT）、规模因子（SMB）、价值因子（HML）等，用于因子归因分析（如FF3模型）。

### 5.2 数据来源
- **商业数据库**：如Wind、Choice、聚源等，数据质量高、覆盖全面，但需要付费。
- **免费数据源**：如新浪财经、东方财富、Tushare Pro等，可获取基础量价和财务数据，适合学生或个人投资者。
- **交易所/公司公告**：官方发布的原始数据，如上市公司财报、分红送配信息等，可用于验证数据准确性。

### 5.3 数据清洗
- **缺失值处理**：对于停牌、未上市等原因导致的数据缺失，可采用前向填充（forward fill）、后向填充（backward fill）或删除缺失样本。
- **异常值处理**：使用3σ法则、分位数法等识别并处理异常值（如极端涨跌停、数据录入错误）。
- **复权处理**：由于分红、送配会影响股价，需进行前复权（保持当前价格不变，调整历史价格）或后复权（保持历史价格不变，调整当前价格），确保价格序列的连续性。
- **停牌处理**：对于停牌期间的收益率，通常按0处理或使用市场基准收益率替代。

### 5.4 数据预处理
- **频率转换**：将日频数据转换为周频、月频等，以匹配策略的调仓周期（如月频调仓策略使用月频数据）。
- **标准化**：对因子进行标准化（如Z-score标准化），消除量纲差异，便于因子比较和组合构建。
- **时间对齐**：财务数据通常滞后发布（如年报在次年4月底前发布），需调整时间戳以避免数据泄露（如使用T-1期财务数据预测T期收益率）。

### 5.5 因子计算
根据策略逻辑计算所需因子，常见传统因子包括：
- **动量因子**：如过去12个月（剔除最近1个月）的收益率。
- **价值因子**：如市盈率（P/E）、市净率（P/B）、市销率（P/S）等。
- **规模因子**：如总市值、流通市值。
- **质量因子**：如ROE、净利润增长率、资产收益率（ROA）等。
- **波动率因子**：如过去3个月的日收益率标准差。

### 5.6 数据存储与管理
- **文件存储**：使用CSV、Excel等格式存储清洗后的数据，便于查看和手动处理。
- **数据库存储**：使用SQLite、MySQL等数据库管理结构化数据，提高查询和处理效率。
- **数据版本控制**：对数据处理流程进行记录，确保策略可复现性。

### 5.7 数据验证
- **一致性检查**：验证不同来源数据的一致性（如Wind与Tushare数据的对比）。
- **完整性检查**：确保数据覆盖策略测试的完整时间区间（如回测期间的所有股票数据）。
- **准确性检查**：验证关键指标的计算是否正确（如复权价格、收益率计算）。
